# Production Line Optimization Simulation

This project simulates a simple production line to identify bottlenecks and test improvement strategies. The simulation is built using Python and the SimPy library.

## Project Goal

The main goal of this project is to use discrete-event simulation to analyze and optimize a manufacturing production line. We will:

1.  Model a multi-stage production process.
2.  Identify bottleneck machines.
3.  Test various improvement scenarios.
4.  Quantify the impact of these changes on key performance indicators (KPIs) like throughput and machine utilization.

## Getting Started

To run this simulation, you will need Python and the SimPy library.

```bash
pip install simpy
```

Then, you can run the simulation with:

```bash
python production_line.py
```
